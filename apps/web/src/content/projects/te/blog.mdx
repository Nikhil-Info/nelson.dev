నా వ్యక్తిగత వెబ్సైట్ మరియు బ్లాగ్.

నేను ఇంటర్నెట్లో చాలా వెబ్సైట్లను చూస్తున్నాను. బ్లాగ్, నేను నా స్వంత బ్లాగ్ను సృష్టించాలనుకుంటున్నాను, అక్కడ నేను నా ఇష్టం వచ్చినది వ్రాయగలను. ట్యుటోరియల్, నోట్స్ లేదా జర్నల్ లేకుండా. అందువల్ల, నేను డిసెంబర్ 5, 2020న వెబ్ డెవలప్మెంట్ నైపుణ్యాలను నేర్చుకోవడం ప్రారంభించాను. మొదట, నేను HTML, CSS మరియు JavaScript నేర్చుకున్నాను. ఒక సంవత్సరం తర్వాత, స్టాటిక్ వెబ్సైట్ నాకు సరిపోదని గ్రహించాను. మరియు నేను React.js నేర్చుకోవడం ప్రారంభించాను, తర్వాత Next.js.
