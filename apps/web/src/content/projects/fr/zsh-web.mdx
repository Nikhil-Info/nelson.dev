J'ai beaucoup d'idées folles, et l'une d'elles est d'exécuter Zsh dans le navigateur. J'utilise Zsh depuis longtemps et je l'adore. Donc, j'essaie de créer Zsh Web. C'est un terminal basé sur le web qui exécute Zsh simultanément dans le navigateur. Il est construit avec Next.js et TypeScript.
