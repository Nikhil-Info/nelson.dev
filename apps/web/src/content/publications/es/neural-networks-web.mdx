## Resumen

Esta investigación explora la aplicación de modelos de redes neuronales para identificar y remediar automáticamente problemas de accesibilidad web, proporcionando retroalimentación en tiempo real a desarrolladores y creadores de contenido.

## Metodología

Entrenamos modelos basados en transformers en grandes conjuntos de datos de componentes web accesibles e inaccesibles para predecir puntajes de cumplimiento WCAG.

## Resultados

Nuestro modelo logró un 94% de precisión en la identificación de violaciones de accesibilidad y proporcionó sugerencias accionables para la mejora.

## Impacto

Este enfoque reduce significativamente el tiempo y la experiencia requerida para crear experiencias web accesibles.
