## МОЙ ПУТЬ

Я Full Stack инженер, который создает производительные, масштабируемые приложения с использованием TypeScript и Next.js. Когда я не программирую, вы найдете меня погруженным в стратегические игры - оба разделяют одни и те же основные принципы: сложное решение проблем, перспективное мышление и удовлетворение от выполнения идеального плана.

Мой путь в программировании начался 5 декабря 2015 года, когда мне было всего 17 лет. То, что началось как хобби, превратилось в страсть, изучая полностью через самообучение на YouTube. Особая благодарность [The Net Ninja](https://www.youtube.com/@NetNinja) - его уроки были ключевыми в построении моей основы в веб-разработке.

Я построил свой рабочий процесс вокруг современных инструментов, которые я люблю: создание опыта с [Next.js](https://nextjs.org/), сотрудничество и версионирование с [GitHub](https://github.com), и беспрепятственное развертывание с [Vercel](https://vercel.com).

## ПОСТРОЕНО С

Этот сайт представляет мою приверженность использованию лучших современных инструментов для создания быстрых, надежных и поддерживаемых веб-впечатлений.

**Основной стек**
- **Фреймворк**: [Next.js](https://nextjs.org/) - React фреймворк для продакшена
- **База данных**: [PostgreSQL](https://www.postgresql.org/) - Надежная реляционная база данных
- **ORM**: [Drizzle](https://orm.drizzle.team/) - TypeScript-first ORM
- **Аутентификация**: [Better Auth](https://www.better-auth.com/) - Современная аутентификация
- **Стилизация**: [Tailwind CSS](https://tailwindcss.com) - Utility-first CSS фреймворк

**Контент и аналитика**
- **Контент**: [MDX](https://mdxjs.com/) - Markdown с JSX компонентами
- **Подсветка синтаксиса**: [Shiki](https://github.com/shikijs/shiki) - Красивые блоки кода
- **Аналитика**: [Umami](https://github.com/umami-software/umami) - Аналитика с фокусом на конфиденциальность
- **Развертывание**: [Vercel](https://vercel.com) - Оптимальная платформа для Next.js
