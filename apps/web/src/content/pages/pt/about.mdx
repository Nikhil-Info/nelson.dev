## MINHA JORNADA

Sou um Engenheiro Full Stack que constrói aplicações performáticas e escaláveis usando TypeScript e Next.js. Quando não estou programando, você me encontrará imerso em jogos estratégicos - ambos compartilham os mesmos princípios fundamentais: resolução complexa de problemas, pensamento à frente e a satisfação de executar um plano perfeito.

Minha jornada de programação começou em 5 de dezembro de 2015, quando eu tinha apenas 17 anos. O que começou como um hobby evoluiu para uma paixão, aprendendo inteiramente através de autoestudo no YouTube. Um agradecimento especial ao [The Net Ninja](https://www.youtube.com/@NetNinja) - seus tutoriais foram fundamentais para construir minha base em desenvolvimento web.

Construí meu fluxo de trabalho em torno de ferramentas modernas que amo: criando experiências com [Next.js](https://nextjs.org/), colaborando e versionando com [GitHub](https://github.com), e implantando perfeitamente com [Vercel](https://vercel.com).

## CONSTRUÍDO COM

Este site representa meu compromisso de usar as melhores ferramentas modernas para criar experiências web rápidas, confiáveis e sustentáveis.

**Stack Principal**
- **Framework**: [Next.js](https://nextjs.org/) - Framework React para produção
- **Banco de dados**: [PostgreSQL](https://www.postgresql.org/) - Banco de dados relacional robusto
- **ORM**: [Drizzle](https://orm.drizzle.team/) - ORM com foco em TypeScript
- **Autenticação**: [Better Auth](https://www.better-auth.com/) - Autenticação moderna
- **Estilização**: [Tailwind CSS](https://tailwindcss.com) - Framework CSS utility-first

**Conteúdo e Analytics**
- **Conteúdo**: [MDX](https://mdxjs.com/) - Markdown com componentes JSX
- **Realce de sintaxe**: [Shiki](https://github.com/shikijs/shiki) - Blocos de código bonitos
- **Analytics**: [Umami](https://github.com/umami-software/umami) - Analytics com foco em privacidade
- **Implantação**: [Vercel](https://vercel.com) - Plataforma ideal para Next.js
