## எனது பயணம்

நான் TypeScript மற்றும் Next.js ஐப் பயன்படுத்தி செயல்திறன் மிக்க, அளவிடக்கூடிய பயன்பாடுகளை உருவாக்கும் ஒரு ஃபுல் ஸ்டாக் இன்ஜினியர். நான் கோடிங் செய்யாத போது, நீங்கள் என்னை உத்தி விளையாட்டுகளில் ஈடுபட்டிருப்பதைக் காணலாம் - இரண்டும் ஒரே முக்கிய கோட்பாடுகளைப் பகிர்ந்து கொள்கின்றன: சிக்கலான சிக்கல் தீர்வு, முன்னோக்கி சிந்தனை மற்றும் ஒரு சரியான திட்டத்தை செயல்படுத்தும் திருப்தி.

எனது கோடிங் பயணம் டிசம்பர் 5, 2015 அன்று தொடங்கியது, நான் வெறும் 17 வயதில் இருந்தபோது. ஒரு பொழுதுபோக்காக தொடங்கியது ஒரு ஆர்வமாக உருவெடுத்தது, YouTube இல் முழுமையாக சுய-படிப்பின் மூலம் கற்றுக்கொண்டேன். [The Net Ninja](https://www.youtube.com/@NetNinja)க்கு சிறப்பு நன்றி - வலை அபிவிருத்தியில் எனது அடித்தளத்தை உருவாக்குவதில் அவரது பயிற்சிகள் முக்கிய பங்கு வகித்தன.

நான் விரும்பும் நவீன கருவிகளைச் சுற்றி எனது பணிப்பாய்வை உருவாக்கியுள்ளேன்: [Next.js](https://nextjs.org/) உடன் அனுபவங்களை உருவாக்குதல், [GitHub](https://github.com) உடன் ஒத்துழைத்தல் மற்றும் பதிப்பாக்கம், மற்றும் [Vercel](https://vercel.com) உடன் தடையின்றி வெளியிடுதல்.

## கட்டப்பட்டது

இந்த தளம் வேகமான, நம்பகமான மற்றும் பராமரிக்கக்கூடிய வலை அனுபவங்களை உருவாக்க சிறந்த நவீன கருவிகளைப் பயன்படுத்துவதற்கான எனது உறுதிப்பாட்டைப் பிரதிபலிக்கிறது.

**முக்கிய ஸ்டாக்**
- **கட்டமைப்பு**: [Next.js](https://nextjs.org/) - உற்பத்திக்கான React கட்டமைப்பு
- **தரவுத்தளம்**: [PostgreSQL](https://www.postgresql.org/) - வலுவான தொடர்பு தரவுத்தளம்
- **ORM**: [Drizzle](https://orm.drizzle.team/) - TypeScript-முதலில் ORM
- **அங்கீகாரம்**: [Better Auth](https://www.better-auth.com/) - நவீன அங்கீகாரம்
- **ஸ்டைலிங்**: [Tailwind CSS](https://tailwindcss.com) - பயன்பாடு-முதலில் CSS கட்டமைப்பு

**உள்ளடக்கம் மற்றும் பகுப்பாய்வு**
- **உள்ளடக்கம்**: [MDX](https://mdxjs.com/) - JSX கூறுகளுடன் Markdown
- **இலக்கணம் சிறப்பித்தல்**: [Shiki](https://github.com/shikijs/shiki) - அழகான குறியீடு தொகுதிகள்
- **பகுப்பாய்வு**: [Umami](https://github.com/umami-software/umami) - தனியுரிமை-மையப்படுத்தப்பட்ட பகுப்பாய்வு
- **வெளியீடு**: [Vercel](https://vercel.com) - Next.jsக்கு உகந்த தளம்
