## MON PARCOURS

Je suis un Ingénieur Full Stack qui construit des applications performantes et évolutives en utilisant TypeScript et Next.js. Quand je ne code pas, vous me trouverez plongé dans des jeux stratégiques - les deux partagent les mêmes principes fondamentaux : résolution complexe de problèmes, pensée prospective et la satisfaction d'exécuter un plan parfait.

Mon parcours de codage a commencé le 5 décembre 2015, alors que je n'avais que 17 ans. Ce qui a commencé comme un hobby s'est transformé en passion, apprenant entièrement par auto-formation sur YouTube. Un remerciement spécial à [The Net Ninja](https://www.youtube.com/@NetNinja) - ses tutoriels ont été déterminants pour construire mes bases en développement web.

J'ai construit mon flux de travail autour d'outils modernes que j'aime : créer des expériences avec [Next.js](https://nextjs.org/), collaborer et versionner avec [GitHub](https://github.com), et déployer de manière transparente avec [Vercel](https://vercel.com).

## CONSTRUIT AVEC

Ce site représente mon engagement à utiliser les meilleurs outils modernes pour créer des expériences web rapides, fiables et maintenables.

**Stack Principal**
- **Framework**: [Next.js](https://nextjs.org/) - Framework React pour la production
- **Base de données**: [PostgreSQL](https://www.postgresql.org/) - Base de données relationnelle robuste
- **ORM**: [Drizzle](https://orm.drizzle.team/) - ORM axé sur TypeScript
- **Authentification**: [Better Auth](https://www.better-auth.com/) - Authentification moderne
- **Stylisation**: [Tailwind CSS](https://tailwindcss.com) - Framework CSS utility-first

**Contenu et Analytics**
- **Contenu**: [MDX](https://mdxjs.com/) - Markdown avec composants JSX
- **Coloration syntaxique**: [Shiki](https://github.com/shikijs/shiki) - Blocs de code magnifiques
- **Analytics**: [Umami](https://github.com/umami-software/umami) - Analytics axé sur la confidentialité
- **Déploiement**: [Vercel](https://vercel.com) - Plateforme optimale pour Next.js
