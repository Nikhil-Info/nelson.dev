## MI VIAJE

Soy un Ingeniero Full Stack que construye aplicaciones performantes y escalables usando TypeScript y Next.js. Cuando no estoy programando, me encontrarás inmerso en juegos estratégicos, ambos comparten los mismos principios fundamentales: resolución compleja de problemas, pensamiento a futuro y la satisfacción de ejecutar un plan perfecto.

Mi viaje de programación comenzó el 5 de diciembre de 2015, cuando solo tenía 17 años. Lo que comenzó como un hobby ha evolucionado hacia una pasión, aprendiendo completamente mediante autoestudio en YouTube. Un agradecimiento especial a [The Net Ninja](https://www.youtube.com/@NetNinja): sus tutoriales fueron fundamentales para construir mi base en desarrollo web.

He construido mi flujo de trabajo alrededor de herramientas modernas que amo: creando experiencias con [Next.js](https://nextjs.org/), colaborando y versionando con [GitHub](https://github.com), y desplegando sin problemas con [Vercel](https://vercel.com).

## CONSTRUIDO CON

Este sitio representa mi compromiso de usar las mejores herramientas modernas para crear experiencias web rápidas, confiables y mantenibles.

**Stack Principal**
- **Framework**: [Next.js](https://nextjs.org/) - Framework React para producción
- **Base de datos**: [PostgreSQL](https://www.postgresql.org/) - Base de datos relacional robusta
- **ORM**: [Drizzle](https://orm.drizzle.team/) - ORM con prioridad en TypeScript
- **Autenticación**: [Better Auth](https://www.better-auth.com/) - Autenticación moderna
- **Estilos**: [Tailwind CSS](https://tailwindcss.com) - Framework CSS utility-first

**Contenido y Analytics**
- **Contenido**: [MDX](https://mdxjs.com/) - Markdown con componentes JSX
- **Resaltado de sintaxis**: [Shiki](https://github.com/shikijs/shiki) - Bloques de código hermosos
- **Analytics**: [Umami](https://github.com/umami-software/umami) - Analytics centrado en privacidad
- **Despliegue**: [Vercel](https://vercel.com) - Plataforma óptima para Next.js
