## Prefácio

Recentemente, reuni uma coleção de sites web excepcionais em design UI/UX que me deixaram maravilhado na minha exploração inicial. Acredito que estes sites web podem enriquecer muito suas habilidades de UI. Aqui está uma lista dos meus 8 sites web favoritos (sem ordem particular).

## Raycast

Raycast é uma ferramenta de produtividade que permite completar tarefas rapidamente. Sua página inicial me cativou com seu design UI, apresentando meus gradientes de cores favoritos e vários tons de vermelho (a cor do tema).

Ao rolar para baixo mostra suas características únicas, demonstrando claramente suas capacidades de produto. Isso me levou a baixar prontamente o software após comprar um [MacBook Air 15"](/uses). O esforço colocado na criação de imagens, design UI e seleção de cores por seus desenvolvedores é evidente no site web e vale a pena aprender com isso.

## Linear

Linear é uma poderosa ferramenta de gerenciamento de projetos projetada para designers e equipes. Oferece uma interface intuitiva e uma gama de características para organizar e rastrear projetos de design. Seja projetando novos produtos, atualizando os existentes, lidando com feedback ou tarefas, Linear fornece ferramentas de colaboração e fluxos de trabalho para garantir execução eficiente de projetos. Através do Linear, equipes de design podem colaborar melhor, rastrear progresso e garantir gerenciamento eficaz em cada etapa.

As animações e o uso de introduções de características de produto em estilo de ladrilho na página inicial são envolventes, e o efeito de fundo desfocado é cativante.

## Figma

Figma não precisa de introdução - é um software profissional de design UI/UX, e seu site web é igualmente impressionante. Seja para pequenos projetos ou grandes equipes, Figma facilita designers alcançarem criatividade enquanto mantêm fluxos de trabalho de colaboração suaves.

O site web emprega várias animações, e sua página inicial é atraente e funcional, oferecendo um ponto de entrada intuitivo para usuários entenderem o valor do Figma.

## Next.js

Next.js é um framework React popular projetado para desenvolvimento moderno de sites web e aplicações. Ele enfatiza desempenho e escalabilidade, fornecendo poderosas capacidades de renderização do lado do servidor e geração de sites web estáticos que otimizam velocidade de carregamento de páginas web e desempenho SEO. Seja construindo aplicações dinâmicas ou sites web de conteúdo estático, Next.js simplifica processos de desenvolvimento e oferece experiências de usuário superiores.

O design minimalista do site web oficial do Next.js é sofisticado, e a parte mais interessante é a seção "Powered By" - uma "CPU" criada usando CSS, surpreendente mas eficaz.

## PlanetScale

PlanetScale é uma plataforma inovadora focada em gerenciamento de banco de dados. Oferece ferramentas e funcionalidades modernas que ajudam equipes de desenvolvimento a projetar, implantar e manter bancos de dados efetivamente. A singularidade do PlanetScale está em sua facilidade de uso e capacidades de automação, aliviando o fardo do gerenciamento de banco de dados e permitindo que desenvolvedores se concentrem em criar aplicações de alta qualidade.

O uso de gradientes, imagens ricas e ilustrações no site web é atraente, e minha parte favorita é o "Tacômetro" no meio. Uma olhada nas ferramentas de desenvolvedor revela como eles usaram CSS para criar este efeito intrigante.

## Fig

Fig é uma ferramenta de linha de comando que melhora significativamente a eficiência com autocompletar e escrita simplificada de scripts, otimizando fluxos de trabalho. Possui uma biblioteca de plugins poderosa e rica onde usuários podem instalar vários plugins convenientes.

O site web emprega numerosos elementos de gradiente, capturando efetivamente a atenção do usuário.

## Pipe

Pipe é uma plataforma de capital moderna que permite que empresas convertam receita futura em capital antecipado. Conexões de dados seguras e em tempo real simplificam o processo de obtenção de capital operacional para empresas. Não há diluição ou obstáculos tradicionais de financiamento. Seu capital disponível depende completamente de sua renda e saúde empresarial. Simplesmente registre, conecte e acesse fundos através do Pipe.

A página inicial exibe efeitos especiais impressionantes, e o design de imagem, tamanho da fonte e interface são visualmente atraentes.

## Framer

Framer é uma poderosa ferramenta de design interativo e prototipagem criada para designers e desenvolvedores. Oferece uma interface intuitiva e uma riqueza de elementos interativos, permitindo que usuários criem facilmente protótipos altamente interativos e designs de interface. Framer integra várias características, incluindo animações, gestos, microinterações e edição de código, permitindo que usuários realizem criativamente seus conceitos de design. Seja para aplicações móveis ou design web, Framer fornece ferramentas excepcionais para ajudar usuários a criar experiências interativas cativantes.

As seções interativas no site web, juntamente com o esquema de cores e animações, refletem um alto nível de profissionalismo.
