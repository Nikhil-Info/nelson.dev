## ஒரு வலைப்பதிவை எவ்வாறு உருவாக்குவது

முதலில், பின்வரும் கட்டளை மூலம் Next.js திட்டத்தை உருவாக்குகிறோம்:

```bash title='முனையம்'
yarn create next-app nextjs-mdx-blog
```

அடுத்து, பின்வரும் கோப்புகளை உருவாக்கவும்:

- `components/layout.jsx` - அனைத்து கூறுகளையும் ஒரு கொள்கலனில் போர்த்தவும் (விருப்பமானது, பாணிக்காக மட்டுமே)
- `date/blog/*.mdx` - வலைப்பதிவு கட்டுரைகள்
- `lib/format-date.js` - தேதியை `மாதம் DD, YYYY` ஆக வடிவமைக்கவும்
- `pages/blog/[slug].jsx` - கட்டுரை பக்கம், [டைனமிக் ரூட்டுகளை](https://nextjs.org/docs/pages/building-your-application/routing/dynamic-routes) பயன்படுத்துகிறது

### மார்க்டவுன் கோப்புகளை எவ்வாறு கையாளுவது

முதலில் `const root{:js}` - ரூட் கோப்பகம், மற்றும் `process.cwd(){:js}` முறை Node.js செயல்முறையின் தற்போதைய பணி கோப்பகத்தை திருப்பித் தரும்.

```js title='lib/mdx.js'
const root = process.cwd()
```

மற்றொரு மாறி `POSTS_PATH{:.entity.name.constant}` கட்டுரை கோப்புகள் சேமிக்கப்படும் பாதைக்கு.

```js title='lib/mdx.js'
import path from 'path'

const POSTS_PATH = path.join(root, 'data', 'blog') // [!code highlight]
// வெளியீடு: A:/nextjs-mdx-blog/data/blog
```

பின்னர் அந்த கோப்பகத்தின் உள்ளடக்கங்களைப் படிக்க fs ஐப் பயன்படுத்தவும், அதாவது `data/blog` கீழ் உள்ள அனைத்து கோப்பு பெயர்களும்.

```js title='lib/mdx.js'
import fs from 'fs'

export const allSlugs = fs.readdirSync(POSTS_PATH)
// வெளியீடு: ['markdown.mdx', 'nextjs.mdx', 'react.mdx']
```

பின்னர் கோப்பு நீட்டிப்பை அகற்ற ஒரு செயல்பாட்டை எழுதவும், இது பின்னர் பயன்படுத்தப்படும்.

```js title='lib/mdx.js'
export const formatSlug = (slug) => slug.replace(/\.mdx$/, '')
/*
 * எடுத்துக்காட்டு: formatSlug('markdown.mdx')
 * வெளியீடு: 'markdown'
 */
```

அடுத்த படி slug மூலம் கட்டுரை உள்ளடக்கத்தைப் பெறுவது.

```js title='lib/mdx.js'
export const getPostBySlug = async (slug) => {
  const postFilePath = path.join(POSTS_PATH, `${slug}.mdx`)
  // வெளியீடு: A:/nextjs-mdx-blog/data/blog/slug.mdx
  const source = fs.readFileSync(postFilePath)
  // கோப்பு உள்ளடக்கத்தை திருப்பி விடு

  const { content, data } = matter(source)
  /*
   * எடுத்துக்காட்டு:
   *  ---
   *  title: வணக்கம்
   *  slug: home
   *  ---
   *  <h1>வணக்கம் உலகம்!</h1>
   *
   * திரும்பப் பெறுதல்:
   *  {
   *    content: '<h1>வணக்கம் உலகம்!</h1>',
   *    data: {
   *      title: 'வணக்கம்',
   *      slug: 'home'
   *    }
   *  }
   */

  const mdxSource = await serialize(content)
  // உள்ளடக்கத்தை தொடர்ச்சியாக்க (next-mdx-remote) கையாள கொடு

  const frontMatter = {
    ...data,
    slug
  }
  // slug ஐ முன் விவரங்களிலும் வைக்கவும், இது பின்னர் பயன்படுத்தப்படும்

  return {
    source: mdxSource,
    frontMatter
  }
}
```

பின்னர் முகப்புப் பக்கத்தில் காட்டப்பட வேண்டிய அனைத்து கட்டுரைகளையும் நீங்கள் பெறலாம்.

```js title='lib/mdx.js'
export const getAllPosts = () => {
  const frontMatter = []

  allSlugs.forEach((slug) => {
    const source = fs.readFileSync(path.join(POSTS_PATH, slug), 'utf-8')

    const { data } = matter(source)

    frontMatter.push({
      ...data,
      slug: formatSlug(slug),
      date: new Date(data.date).toISOString()
    })
  })

  return frontMatter.sort((a, b) => dateSortDesc(a.date, b.date))
}

// தேதியின்படி இறங்கு வரிசையில் வரிசைப்படுத்தப்பட்டது
const dateSortDesc = (a, b) => {
  if (a > b) return -1
  if (a < b) return 1

  return 0
}
```

### தேதியை வடிவமைத்தல்

```js title='lib/format-date.js'
export const formatDate = (date) =>
  new Date(date).toLocaleDateString('ta', {
    year: 'numeric',
    month: 'long',
    day: 'numeric'
  })
/*
 * formatDate('2022-08-21T00:00:00Z')
 * வெளியீடு: 'ஆகஸ்ட் 21, 2022'
 */
```

### முகப்புப் பக்கம்

```jsx title='pages/index.jsx'
import { formatDate } from '../lib/format-date'
import { getAllPosts } from '../lib/mdx'

import Link from 'next/link'

export default function Home({ posts }) {
  return (
    <>
      <h1 className='mb-8 text-6xl font-bold'>வலைப்பதிவு</h1>
      <hr className='my-8' />
      <ul className='flex flex-col gap-3'>
        {posts.map(({ slug, title, summary, date }) => (
          <li key={slug}>
            <Link href={`/blog/${slug}`}>
              <a className='block rounded-lg border border-solid border-gray-300 p-6 shadow-md'>
                <div className='flex justify-between'>
                  <h2>{title}</h2>
                  <time dateTime={date}>{formatDate(date)}</time>
                </div>
                <p className='mt-4'>{summary}</p>
              </a>
            </Link>
          </li>
        ))}
      </ul>
    </>
  )
}

// அனைத்து கட்டுரைகளையும் பெற getStaticProps ஐப் பயன்படுத்தவும்
export const getStaticProps = async () => {
  const posts = getAllPosts()

  return {
    props: {
      posts
    }
  }
}
```

### கட்டுரை பக்கம்

```jsx title='pages/[slug].jsx'
import { formatDate } from '../../lib/format-date'
import { allSlugs, formatSlug, getPostBySlug } from '../../lib/mdx'

import { MDXRemote } from 'next-mdx-remote'

export default function Blog({ post }) {
  const { title, date } = post.frontMatter

  return (
    <div>
      <h1 className='mb-2 text-6xl font-bold'>{title}</h1>
      <time dateTime={date} className='text-lg font-medium'>
        {formatDate(date)}
      </time>
      <hr className='my-8' />
      <article className='prose max-w-none'>
        <MDXRemote {...post.source} />
      </article>
    </div>
  )
}

export const getStaticProps = async ({ params }) => {
  const post = await getPostBySlug(params.slug)

  return {
    props: {
      post
    }
  }
}

export const getStaticPaths = async () => {
  const paths = allSlugs.map((slug) => ({
    params: {
      slug: formatSlug(slug)
    }
  }))
  /*
   * paths வெளியீடு:
   *   [
   *     { params: { slug: 'markdown' } },
   *     { params: { slug: 'nextjs' } },
   *     { params: { slug: 'react' } }
   *   ]
   */

  return {
    paths,
    fallback: false
  }
}
```
