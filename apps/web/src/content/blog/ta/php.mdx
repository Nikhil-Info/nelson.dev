## PHP அறிமுகம்

PHP (முழு பெயர்: PHP: _ஹைபர்டெக்ஸ்ட் பிரீப்ரோசெசர்_) ஒரு திறந்த மூல பொது நோக்க கணினி ஸ்கிரிப்டிங் மொழியாகும், இது வலை மேம்பாட்டிற்கு பொருத்தமானது மற்றும் HTML இல் உட்பொதிக்கப்படுகிறது. PHP இன் தொடரியல் C மொழி, Java மற்றும் Perl போன்ற பிரபலமான கணினி மொழிகளின் பண்புகளை எடுத்துக்கொள்கிறது மற்றும் பொது நிரலாக்கலாளர்களுக்கு கற்றுக்கொள்வது எளிதானது. PHP இன் முக்கிய குறிக்கோள் வலை மேம்பாட்டாளர்கள் விரைவாக டைனமிக் பக்கங்களை எழுத அனுமதிப்பதாகும், ஆனால் PHP பல பிற பகுதிகளிலும் பயன்படுத்தப்படுகிறது.

PHP கோப்பு நீட்டிப்பு `.php`, எ.கா `index.php`. மேலே குறிப்பிட்டுள்ளபடி, `PHP` நேரடியாக `HTML` ஐ உட்பொதிக்க முடியும்.

## பயன்பாடு

உலாவி மேலிருந்து கீழாக `<?php` வாசிக்கும் போது, அது `PHP` ஐ இயக்குகிறது மற்றும் `?>` உடன் முடிகிறது. எடுத்துக்காட்டாக:

```php
<!DOCTYPE html>
<html>
    <head>
        <title>PHP</title>
    </head>
    <body>
        <?php
            echo "இது ஒரு வாக்கியம்."
        ?>
    </body>
</html>
```

## எச்சரிக்கைகள்

`PHP` இல், `PHP` வழிமுறை முடிந்த பிறகு வேறு `HTML` இல்லை. `?>` உடன் முடிக்காமல் இருப்பது பரிந்துரைக்கப்படுகிறது. அது மூடப்பட்டால் சிக்கலை ஏற்படுத்தும்.

```php
<?php
    echo "PHP முடிந்த பிறகு HTML இல்லை";
```

## PHP கருத்துகள்

கிட்டத்தட்ட பெரும்பாலான வலை நிரலாக்க மொழிகள் கருத்து தெரிவிக்க முடியும், மற்றும் `PHP` இன் **ஒற்றை-வரி** முறை `//` மற்றும் **பல-வரி** முறை `/* இங்கே சில உரை */`.

```php
<?php
    echo = "இது ஒரு ஒற்றை வரி கருத்து"; // நான் ஒரு ஒற்றை வரி கருத்து
    echo = "இது ஒரு பல-வரி கருத்து";
    /*
        நான் ஒரு பல-வரி கருத்து
        இரண்டாவது வரி
    */
?>
```

## எக்கோ

`PHP` இல் எளிய தொடரியல் `Echo`. `Echo` இன் உள்ளடக்கம் இரட்டை மேற்கோள்கள் `""` அல்லது ஒற்றை மேற்கோள்கள் `''` இல் இருக்க வேண்டும். ஒவ்வொரு தொடரியலும் அரைப்புள்ளி `;` உடன் முடிய வேண்டும்.

```php
<?php
    echo "இது இரட்டை மேற்கோள்களை பயன்படுத்தும் எடுத்துக்காட்டு.";
    echo 'இது ஒற்றை மேற்கோள்களை பயன்படுத்தும் எடுத்துக்காட்டு.';
?>
```

`Echo` இல் உள்ள உள்ளடக்கம் `HTML` தொடரியலையும் பயன்படுத்தலாம். எடுத்துக்காட்டாக, நான் இங்கே `<br/>` பயன்படுத்தினேன், அதனால் வரி முறிக்கப்படும்.

```php
<?php
    echo "இது முதல் வாக்கியம்<br/>";
    echo "இது இரண்டாவது வாக்கியம்";
    echo "<a href=\"https://example.com\">இங்கே ஒரு இணைப்பு உள்ளது</a>";
?>
```

## மாறி

`PHP` எழுதுவதை எளிதாக்க `Javascript` போன்ற ஒரு மாறிக்கு மதிப்பை ஒதுக்கலாம். நாம் டாலர் அடையாளம் `$` பயன்படுத்துவோம்.

```php
<?php
    $மதிப்பு = "நான் ஒரு மதிப்பு";
?>
```

மாறிகளுக்கு பெயரிடும் போது பின்பற்ற வேண்டிய சில விதிகள் உள்ளன.

<அட்டவணை
  headers={['எடுத்துக்காட்டு', 'விளக்கம்', 'விதிகள்']}
  rows={[
    ['$மதிப்பு', 'சிற்றெழுத்து', '✅'],
    ['$மதிப்பு', 'பேரெழுத்து', '✅'],
    ['$_மதிப்பு', 'அண்டர்ஸ்கோர்', '✅'],
    ['$1மதிப்பு', 'எண்ணில் இருந்து தொடங்கு', '❌']
  ]}
/>

மாறி பெயர்களில் `பேரெழுத்து` மற்றும் `சிற்றெழுத்து` இடையே வேறுபாடு உள்ளது. பேரெழுத்து மாறிகள் சிற்றெழுத்து மாறிகளுக்கு சமமாக இல்லை.

## மாறி பெயரிடும் உதவிக்குறிப்புகள்

### கேமல் வழக்கு

ஒரு மாறி பெயர் மற்றும் செயல்பாடு பெயர் இரண்டு அல்லது அதற்கு மேற்பட்ட சொற்களால் ஒரு தனித்துவமான அடையாள சொல்லாக இணைக்கப்படும் போது, "கேமல் வழக்கு" பயன்பாடு மாறிகள் மற்றும் செயல்பாடுகளின் வாசிப்புத்திறனை அதிகரிக்கும். ஏனெனில் மாறிகளில் `இடைவெளிகள்` இருக்க முடியாது.

```php
<?php
    $ஒன்றுபிளஸ்ஒன்றுபதில் = 1 + 1;
    echo $ஒன்றுபிளஸ்ஒன்றுபதில்;
?>
```

### பாம்பு வழக்கு

பாம்பு வழக்கு (பாம்பு_வழக்கு போன்றது) ஒரு எழுத்து பாணியைக் குறிக்கிறது, இதில் ஒவ்வொரு இடைவெளியும் அண்டர்ஸ்கோர் ( \_ ) எழுத்துடன் மாற்றப்படுகிறது மற்றும் ஒவ்வொரு சொல்லின் முதல் எழுத்தும் சிற்றெழுத்தில் எழுதப்படுகிறது.

```php
<?php
    $ஒன்று_பிளஸ்_ஒன்று_பதில் = 1 + 1;
    echo $ஒன்று_பிளஸ்_ஒன்று_பதில்;
?>
```

## சரம் ஆபரேட்டர்

இரண்டு சரம் ஆபரேட்டர்கள் உள்ளன. முதலாவது இணைப்பு ஆபரேட்டர் ("."), இது அதன் இடது மற்றும் வலது வாதங்களின் இணைக்கப்பட்ட சரத்தை திருப்பித் தருகிறது. இரண்டாவது இணைப்பு ஒதுக்கீடு ஆபரேட்டர் (".="), இது வலது வாதத்தை இடது வாதத்தில் சேர்க்கிறது.

```php
<?php
    echo "இது" . " " . "சரம்" . " " . "ஆபரேட்டர்கள்" . ".";
?>
```

## எண்கணித ஆபரேட்டர்

```php
<?php
    $மதிப்பு0 = 1.5 // மிதவை
    $மதிப்பு1 = 1 + 1 // கூட்டல்
    $மதிப்பு2 = 1 - 1 // கழித்தல்
    $மதிப்பு3 = 1 * 1 // பெருக்கல்
    $மதிப்பு4 = 1 / 1 // வகுத்தல்
    $மதிப்பு5 = 12 / 5 // வகுத்தல் | முழுமையற்ற வகுத்தல் மிதவையாக மாறும், இது மற்ற நிரலாக்க மொழிகளிலிருந்து வேறுபட்டது, php மீதியை கணக்கிடும்
    $மதிப்பு6 = 3 % 5 // மீதியை எடுக்கவும்
?>
```
