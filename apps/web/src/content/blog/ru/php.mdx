## Введение в PHP

PHP (полное название: PHP: _Hypertext Preprocessor_) - это язык сценариев общего назначения с открытым исходным кодом, особенно подходящий для веб-разработки и встраиваемый в HTML. Синтаксис PHP заимствует характеристики популярных языков программирования, таких как C, Java и Perl, и легко изучается обычными программистами. Основная цель PHP - позволить веб-разработчикам быстро писать динамические страницы, но PHP также используется во многих других областях.

Расширение файла PHP - `.php`, например `index.php`. Как упоминалось выше, `PHP` может напрямую встраивать `HTML`.

## Использование

Когда браузер читает `<?php` сверху вниз, он выполняет `PHP` и заканчивает `?>`. Например:

```php
<!DOCTYPE html>
<html>
    <head>
        <title>PHP</title>
    </head>
    <body>
        <?php
            echo "Это предложение."
        ?>
    </body>
</html>
```

## Меры предосторожности

В `PHP` нет другого `HTML` после завершения инструкции `PHP`. Рекомендуется не заканчивать `?>`. Это вызовет проблемы, если закрыть.

```php
<?php
    echo "Нет HTML после завершения PHP";
```

## Комментарии PHP

Почти большинство веб-языков программирования могут комментировать, и **однострочный** способ `PHP` - `//`, а **многострочный** способ - `/* Какой-то текст здесь */`.

```php
<?php
    echo = "Это однострочный комментарий"; // Я однострочный комментарий
    echo = "Это многострочный комментарий";
    /*
        Я многострочный комментарий
        Вторая строка
    */
?>
```

## Echo

Самая простая синтаксическая конструкция в `PHP` - `Echo`. Содержимое `Echo` должно быть заключено в двойные кавычки `""` или одинарные кавычки `''`. Каждая синтаксическая конструкция должна заканчиваться точкой с запятой `;`.

```php
<?php
    echo "Это пример использования двойных кавычек.";
    echo 'Это пример использования одинарных кавычек.';
?>
```

Содержимое в `Echo` также может использовать синтаксис `HTML`. Например, я использовал здесь `<br/>`, чтобы строка могла разрываться.

```php
<?php
    echo "Это первое предложение<br/>";
    echo "Это второе предложение";
    echo "<a href=\"https://example.com\">вот ссылка</a>";
?>
```

## Переменная

Мы можем присвоить значение переменной, чтобы облегчить написание `PHP`, аналогично `Javascript`. Мы будем использовать знак доллара `$`.

```php
<?php
    $значение = "Я значение";
?>
```

Есть некоторые правила, которым нужно следовать при именовании переменных.

  headers={['Пример', 'Описание', 'Правила']}

    ['$_значение', 'Подчеркивание', '✅'],

Есть разница между `прописными` и `строчными` в именах переменных. Прописные переменные не равны строчным переменным.

## Советы по именованию переменных

### Camel case

Когда имя переменной и имя функции связаны двумя или более словами, чтобы сформировать уникальное идентификационное слово, использование "camel case" может увеличить читаемость переменных и функций. Потому что мы не можем иметь `пробелы` в переменных.

```php
<?php
    $одинПлюсОдинОтвет = 1 + 1;
    echo $одинПлюсОдинОтвет;
?>
```

### Snake case

Snake case (как snake_case) относится к стилю письма, в котором каждый пробел заменяется символом подчеркивания ( \_ ), а первая буква каждого слова пишется в нижнем регистре.

```php
<?php
    $один_плюс_один_ответ = 1 + 1;
    echo $один_плюс_один_ответ;
?>
```

## Оператор строки

Есть два строковых оператора. Первый - оператор конкатенации ("."), который возвращает объединенную строку своих левого и правого аргументов. Второй - оператор присваивания конкатенации (".="), который добавляет правый аргумент к левому аргументу.

```php
<?php
    echo "Это" . " " . "строковые" . " " . "операторы" . ".";
?>
```

## Арифметический оператор

```php
<?php
    $значение0 = 1.5 // Float
    $значение1 = 1 + 1 // Плюс
    $значение2 = 1 - 1 // Минус
    $значение3 = 1 * 1 // Умножить
    $значение4 = 1 / 1 // Разделить
    $значение5 = 12 / 5 // Разделить | Неполное деление станет float, что отличается от других языков программирования, php вычислит остаток
    $значение6 = 3 % 5 // Взять остаток
?>
```
