## Как создать блог

Сначала создаем проект Next.js с помощью следующей команды:

```bash title='Терминал'
yarn create next-app nextjs-mdx-blog
```

Затем создайте следующие файлы:

- `components/layout.jsx` - Обернуть все компоненты в контейнер (опционально, только для стиля)
- `date/blog/*.mdx` - Статьи блога
- `lib/format-date.js` - Форматировать дату как `DD месяц YYYY`
- `pages/blog/[slug].jsx` - страница статьи, используя [динамические маршруты](https://nextjs.org/docs/pages/building-your-application/routing/dynamic-routes)

### Как обрабатывать Markdown файлы

Сначала `const root{:js}` - корневая директория, и метод `process.cwd(){:js}` возвращает текущую рабочую директорию процесса Node.js.

```js title='lib/mdx.js'
const root = process.cwd()
```

Другая переменная `POSTS_PATH{:.entity.name.constant}` для пути, где хранятся файлы статей.

```js title='lib/mdx.js'
import path from 'path'

const POSTS_PATH = path.join(root, 'data', 'blog') // [!code highlight]
// Вывод: A:/nextjs-mdx-blog/data/blog
```

Затем используйте fs для чтения содержимого этой директории, то есть всех имен файлов в `data/blog`.

```js title='lib/mdx.js'
import fs from 'fs'

export const allSlugs = fs.readdirSync(POSTS_PATH)
// Вывод: ['markdown.mdx', 'nextjs.mdx', 'react.mdx']
```

Затем напишите функцию для удаления расширения файла, которая будет использоваться позже.

```js title='lib/mdx.js'
export const formatSlug = (slug) => slug.replace(/\.mdx$/, '')
/*
 * Пример: formatSlug('markdown.mdx')
 * Вывод: 'markdown'
 */
```

Следующий шаг - получить содержимое статьи по slug.

```js title='lib/mdx.js'
export const getPostBySlug = async (slug) => {
  const postFilePath = path.join(POSTS_PATH, `${slug}.mdx`)
  // Вывод: A:/nextjs-mdx-blog/data/blog/slug.mdx
  const source = fs.readFileSync(postFilePath)
  // Возвращает содержимое файла

  const { content, data } = matter(source)
  /*
   * Пример:
   *  ---
   *  title: Привет
   *  slug: home
   *  ---
   *  <h1>Привет мир!</h1>
   *
   * Возврат:
   *  {
   *    content: '<h1>Привет мир!</h1>',
   *    data: {
   *      title: 'Привет',
   *      slug: 'home'
   *    }
   *  }
   */

  const mdxSource = await serialize(content)
  // Поместить содержимое для сериализации (next-mdx-remote) для обработки

  const frontMatter = {
    ...data,
    slug
  }
  // Поместить slug в front matter тоже, который будет использоваться позже

  return {
    source: mdxSource,
    frontMatter
  }
}
```

Затем вы можете получить все статьи для отображения на домашней странице.

```js title='lib/mdx.js'
export const getAllPosts = () => {
  const frontMatter = []

  allSlugs.forEach((slug) => {
    const source = fs.readFileSync(path.join(POSTS_PATH, slug), 'utf-8')

    const { data } = matter(source)

    frontMatter.push({
      ...data,
      slug: formatSlug(slug),
      date: new Date(data.date).toISOString()
    })
  })

  return frontMatter.sort((a, b) => dateSortDesc(a.date, b.date))
}

// Отсортировано по дате в порядке убывания
const dateSortDesc = (a, b) => {
  if (a > b) return -1
  if (a < b) return 1

  return 0
}
```

### Форматирование даты

```js title='lib/format-date.js'
export const formatDate = (date) =>
  new Date(date).toLocaleDateString('ru', {
    year: 'numeric',
    month: 'long',
    day: 'numeric'
  })
/*
 * formatDate('2022-08-21T00:00:00Z')
 * Вывод: '21 августа 2022 г.'
 */
```

### Домашняя страница

```jsx title='pages/index.jsx'
import { formatDate } from '../lib/format-date'
import { getAllPosts } from '../lib/mdx'

import Link from 'next/link'

export default function Home({ posts }) {
  return (
    <>
      <h1 className='mb-8 text-6xl font-bold'>Блог</h1>
      <hr className='my-8' />
      <ul className='flex flex-col gap-3'>
        {posts.map(({ slug, title, summary, date }) => (
          <li key={slug}>
            <Link href={`/blog/${slug}`}>
              <a className='block rounded-lg border border-solid border-gray-300 p-6 shadow-md'>
                <div className='flex justify-between'>
                  <h2>{title}</h2>
                  <time dateTime={date}>{formatDate(date)}</time>
                </div>
                <p className='mt-4'>{summary}</p>
              </a>
            </Link>
          </li>
        ))}
      </ul>
    </>
  )
}

// Используйте getStaticProps для получения всех статей
export const getStaticProps = async () => {
  const posts = getAllPosts()

  return {
    props: {
      posts
    }
  }
}
```

### Страница статьи

```jsx title='pages/[slug].jsx'
import { formatDate } from '../../lib/format-date'
import { allSlugs, formatSlug, getPostBySlug } from '../../lib/mdx'

import { MDXRemote } from 'next-mdx-remote'

export default function Blog({ post }) {
  const { title, date } = post.frontMatter

  return (
    <div>
      <h1 className='mb-2 text-6xl font-bold'>{title}</h1>
      <time dateTime={date} className='text-lg font-medium'>
        {formatDate(date)}
      </time>
      <hr className='my-8' />
      <article className='prose max-w-none'>
        <MDXRemote {...post.source} />
      </article>
    </div>
  )
}

export const getStaticProps = async ({ params }) => {
  const post = await getPostBySlug(params.slug)

  return {
    props: {
      post
    }
  }
}

export const getStaticPaths = async () => {
  const paths = allSlugs.map((slug) => ({
    params: {
      slug: formatSlug(slug)
    }
  }))
  /*
   * Вывод paths:
   *   [
   *     { params: { slug: 'markdown' } },
   *     { params: { slug: 'nextjs' } },
   *     { params: { slug: 'react' } }
   *   ]
   */

  return {
    paths,
    fallback: false
  }
}
```

Таким образом, простой блог готов.
