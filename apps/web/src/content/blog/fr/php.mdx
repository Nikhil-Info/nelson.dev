## Introduction de PHP

PHP (nom complet: PHP: _Hypertext Preprocessor_) est un langage de script informatique open source à usage général, particulièrement adapté au développement web et intégré dans HTML. La syntaxe de PHP s'inspire des caractéristiques de langages informatiques populaires tels que le langage C, Java et Perl, et est facile à apprendre pour les programmeurs généraux. L'objectif principal de PHP est de permettre aux développeurs web d'écrire rapidement des pages dynamiques, mais PHP est également utilisé dans de nombreux autres domaines.

L'extension de fichier PHP est `.php`, par exemple `index.php`. Comme mentionné ci-dessus, `PHP` peut intégrer directement `HTML`.

## Utilisation

Lorsque le navigateur lit `<?php` de haut en bas, il exécute `PHP` et se termine par `?>`. Par exemple :

```php
<!DOCTYPE html>
<html>
    <head>
        <title>PHP</title>
    </head>
    <body>
        <?php
            echo "Ceci est une phrase."
        ?>
    </body>
</html>
```

## Précautions

Dans `PHP`, il n'y a pas d'autre `HTML` après la fin de l'instruction `PHP`. Il est recommandé de ne pas terminer par `?>`. Cela causera des problèmes si c'est fermé.

```php
<?php
    echo "Pas de HTML après la fin de PHP";
```

## Commentaires PHP

Presque la plupart des langages de programmation web peuvent commenter, et la manière **monoligne** de `PHP` est `//` et la manière **multiligne** est `/* Du texte ici */`.

```php
<?php
    echo = "Ceci est un commentaire monoligne"; // Je suis un commentaire monoligne
    echo = "Ceci est un commentaire multiligne";
    /*
        Je suis un commentaire multiligne
        Deuxième ligne
    */
?>
```

## Echo

La syntaxe la plus simple dans `PHP` est `Echo`. Le contenu de `Echo` doit être entre guillemets doubles `""` ou guillemets simples `''`. Chaque syntaxe doit se terminer par un point-virgule `;`.

```php
<?php
    echo "Ceci est l'exemple utilisant des guillemets doubles.";
    echo 'Ceci est l'exemple utilisant des guillemets simples.';
?>
```

Le contenu dans `Echo` peut également utiliser la syntaxe `HTML`. Par exemple, j'ai utilisé `<br/>` ici, pour que la ligne puisse être rompue.

```php
<?php
    echo "Ceci est la première phrase<br/>";
    echo "Ceci est la deuxième phrase";
    echo "<a href=\"https://example.com\">voici un lien</a>";
?>
```

## Variable

Nous pouvons attribuer une valeur à une variable pour nous faciliter l'écriture de `PHP`, similaire à `Javascript`. Nous utiliserons le signe dollar `$`.

```php
<?php
    $valeur = "Je suis une valeur";
?>
```

Il y a quelques règles à suivre lorsque nous nommons des variables.

Il y a une différence entre `majuscules` et `minuscules` dans les noms de variables. Les variables en majuscules ne sont pas égales aux variables en minuscules.

## Astuces de nommage de variables

### Camel case

Lorsqu'un nom de variable et un nom de fonction sont liés ensemble par deux mots ou plus pour former un mot d'identification unique, l'utilisation du "camel case" peut augmenter la lisibilité des variables et fonctions. Parce que nous ne pouvons pas avoir d'`espaces` dans les variables.

```php
<?php
    $unPlusUnReponse = 1 + 1;
    echo $unPlusUnReponse;
?>
```

### Snake case

Le snake case (comme snake_case) fait référence à un style d'écriture dans lequel chaque espace est remplacé par un caractère de soulignement ( \_ ) et la première lettre de chaque mot est écrite en minuscules.

```php
<?php
    $un_plus_un_reponse = 1 + 1;
    echo $un_plus_un_reponse;
?>
```

## Opérateur de chaîne

Il existe deux opérateurs de chaîne. Le premier est l'opérateur de concaténation ("."), qui renvoie la chaîne concaténée de ses arguments gauche et droit. Le second est l'opérateur de concaténation d'affectation (".="), qui ajoute l'argument droit à l'argument gauche.

```php
<?php
    echo "Ceci" . " " . "est" . " " . "les" . " " . "opérateurs" . " " . "de" . " " . "chaîne" . ".";
?>
```

## Opérateur arithmétique

```php
<?php
    $valeur0 = 1.5 // Float
    $valeur1 = 1 + 1 // Plus
    $valeur2 = 1 - 1 // Moins
    $valeur3 = 1 * 1 // Fois
    $valeur4 = 1 / 1 // Diviser
    $valeur5 = 12 / 5 // Diviser | La division incomplète deviendra float, ce qui est différent des autres langages de programmation, php calculera le reste
    $valeur6 = 3 % 5 // Prendre le reste
?>
```
