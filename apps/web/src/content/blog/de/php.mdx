## Einführung in PHP

PHP (vollständiger Name: PHP: _Hypertext Preprocessor_) ist eine quelloffene allgemeine Computerskriptsprache, die besonders für Webentwicklung geeignet ist und in HTML eingebettet werden kann. Die Syntax von PHP übernimmt Merkmale beliebter Computersprachen wie C, Java und Perl und ist für allgemeine Programmierer leicht zu erlernen. Das Hauptziel von PHP ist es, Webentwicklern zu ermöglichen, schnell dynamische Seiten zu schreiben, aber PHP wird auch in vielen anderen Bereichen eingesetzt.

Die PHP-Dateierweiterung ist `.php`, z.B. `index.php`. Wie oben erwähnt, kann `PHP` direkt `HTML` einbetten.

## Verwendung

Wenn der Browser von oben nach unten `<?php` liest, führt er `PHP` aus und endet mit `?>`. Zum Beispiel:

```php
<!DOCTYPE html>
<html>
    <head>
        <title>PHP</title>
    </head>
    <body>
        <?php
            echo "Dies ist ein Satz."
        ?>
    </body>
</html>
```

## Hinweise

In `PHP` gibt es nach Abschluss der `PHP`-Anweisung kein weiteres `HTML`. Es wird empfohlen, nicht mit `?>` zu enden. Das Schließen kann Probleme verursachen.

```php
<?php
    echo "Kein HTML nach PHP-Ende";
```

## PHP-Kommentare

Fast die meisten Webprogrammiersprachen können kommentieren, und die **einzeilige** Art von `PHP` ist `//` und die **mehrzeilige** Art ist `/* Hier etwas Text */`.

```php
<?php
    echo = "Dies ist ein einzeiliger Kommentar"; // Ich bin ein einzeiliger Kommentar
    echo = "Dies ist ein mehrzeiliger Kommentar";
    /*
        Ich bin ein mehrzeiliger Kommentar
        Zweite Zeile
    */
?>
```

## Echo

Die einfachste Syntax in `PHP` ist `Echo`. Der Inhalt von `Echo` muss in doppelte Anführungszeichen `""` oder einfache Anführungszeichen `''` eingeschlossen werden. Jede Syntax muss mit einem Semikolon `;` enden.

```php
<?php
    echo "Dies ist das Beispiel für die Verwendung doppelter Anführungszeichen.";
    echo 'Dies ist das Beispiel für die Verwendung einfacher Anführungszeichen.';
?>
```

Inhalt in `Echo` kann auch `HTML`-Syntax verwenden. Zum Beispiel habe ich hier `<br/>` verwendet, damit die Zeile umgebrochen werden kann.

```php
<?php
    echo "Dies ist der erste Satz<br/>";
    echo "Dies ist der zweite Satz";
    echo "<a href=\"https://example.com\">hier ist ein Link</a>";
?>
```

## Variable

Wir können einer Variable einen Wert zuweisen, um uns das Schreiben von `PHP` zu erleichtern, ähnlich wie bei `Javascript`. Wir verwenden das Dollarzeichen `$`.

```php
<?php
    $wert = "Ich bin ein Wert";
?>
```

Es gibt einige Regeln, die wir beim Benennen von Variablen beachten müssen.

Es gibt einen Unterschied zwischen `Großbuchstaben` und `Kleinbuchstaben` in Variablennamen. Großgeschriebene Variablen sind nicht gleich kleingeschriebenen Variablen.

## Tipps zur Variablenbenennung

### Camel Case

Wenn ein Variablenname und ein Funktionsname durch zwei oder mehr Wörter zu einem eindeutigen Identifikationswort verbunden sind, kann die Verwendung von "Camel Case" die Lesbarkeit von Variablen und Funktionen erhöhen. Weil wir keine `Leerzeichen` in Variablen haben können.

```php
<?php
    $einsPlusEinsAntwort = 1 + 1;
    echo $einsPlusEinsAntwort;
?>
```

### Snake Case

Snake Case (wie snake_case) bezieht sich auf einen Schreibstil, bei dem jedes Leerzeichen durch einen Unterstrich ( \_ ) ersetzt wird und der erste Buchstabe jedes Wortes klein geschrieben wird.

```php
<?php
    $eins_plus_eins_antwort = 1 + 1;
    echo $eins_plus_eins_antwort;
?>
```

## String-Operator

Es gibt zwei String-Operatoren. Der erste ist der Verkettungsoperator ("."), der die verkettete Zeichenkette seiner linken und rechten Argumente zurückgibt. Der zweite ist der Verkettungszuweisungsoperator (".="), der das rechte Argument an das linke Argument anhängt.

```php
<?php
    echo "Dies" . " " . "ist" . " " . "String" . " " . "Operatoren" . ".";
?>
```

## Arithmetischer Operator

```php
<?php
    $wert0 = 1.5 // Float
    $wert1 = 1 + 1 // Plus
    $wert2 = 1 - 1 // Minus
    $wert3 = 1 * 1 // Mal
    $wert4 = 1 / 1 // Teilen
    $wert5 = 12 / 5 // Teilen | Unvollständige Division wird zu Float, was sich von anderen Programmiersprachen unterscheidet, php berechnet den Rest
    $wert6 = 3 % 5 // Rest nehmen
?>
```
