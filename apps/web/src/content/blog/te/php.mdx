## PHP పరిచయం

PHP (పూర్తి పేరు: PHP: _హైపర్టెక్స్ట్ ప్రీప్రాసెసర్_) ఒక ఓపెన్ సోర్స్ జనరల్-పర్పస్ కంప్యూటర్ స్క్రిప్టింగ్ లాంగ్వేజ్, ప్రత్యేకంగా వెబ్ డెవలప్మెంట్ కోసం అనుకూలంగా ఉంటుంది మరియు HTMLలో ఎంబెడ్ చేయబడుతుంది. PHP సింటాక్స్ C లాంగ్వేజ్, Java మరియు Perl వంటి ప్రసిద్ధ కంప్యూటర్ భాషల లక్షణాలను తీసుకుంటుంది మరియు సాధారణ ప్రోగ్రామర్లకు నేర్చుకోవడానికి సులభం. PHP యొక్క ప్రధాన లక్ష్యం వెబ్ డెవలపర్లు శీఘ్రంగా డైనమిక్ పేజీలను రాయడానికి అనుమతించడం, కానీ PHP అనేక ఇతర రంగాలలో కూడా ఉపయోగించబడుతుంది.

PHP ఫైల్ ఎక్స్టెన్షన్ `.php`, ఉదా `index.php`. పైన చెప్పినట్లుగా, `PHP` నేరుగా `HTML` ను ఎంబెడ్ చేయగలదు.

## ఉపయోగం

బ్రౌజర్ పై నుండి కిందకు `<?php` చదివినప్పుడు, అది `PHP` ను ఎక్సిక్యూట్ చేస్తుంది మరియు `?>` తో ముగుస్తుంది. ఉదాహరణకు:

```php
<!DOCTYPE html>
<html>
    <head>
        <title>PHP</title>
    </head>
    <body>
        <?php
            echo "ఇది ఒక వాక్యం."
        ?>
    </body>
</html>
```

## జాగ్రత్తలు

`PHP` లో, `PHP` ఇన్స్ట్రక్షన్ పూర్తయిన తర్వాత ఇతర `HTML` ఉండదు. `?>` తో ముగించకుండా ఉండాలని సిఫార్సు చేయబడింది. ఇది క్లోజ్ చేస్తే ఇబ్బంది కలిగిస్తుంది.

```php
<?php
    echo "PHP ముగిసిన తర్వాత HTML లేదు";
```

## PHP కామెంట్స్

దాదాపు చాలా వెబ్ ప్రోగ్రామింగ్ భాషలు కామెంట్ చేయగలవు, మరియు `PHP` యొక్క **సింగిల్-లైన్** మార్గం `//` మరియు **మల్టీ-లైన్** మార్గం `/* ఇక్కడ కొంత టెక్స్ట్ */`.

```php
<?php
    echo = "ఇది ఒక సింగిల్ లైన్ కామెంట్"; // నేను ఒక సింగిల్ లైన్ కామెంట్
    echo = "ఇది ఒక మల్టీ-లైన్ కామెంట్";
    /*
        నేను ఒక మల్టీ-లైన్ కామెంట్
        రెండవ లైన్
    */
?>
```

## ఎకో

`PHP` లో సరళమైన సింటాక్స్ `Echo`. `Echo` యొక్క కంటెంట్ డబుల్ కోట్స్ `""` లేదా సింగిల్ కోట్స్ `''` లో ఉండాలి. ప్రతి సింటాక్స్ సెమికోలన్ `;` తో ముగియాలి.

```php
<?php
    echo "ఇది డబుల్ కోట్స్ ఉపయోగించే ఉదాహరణ.";
    echo 'ఇది సింగిల్ కోట్స్ ఉపయోగించే ఉదాహరణ.';
?>
```

`Echo` లోని కంటెంట్ `HTML` సింటాక్స్ను కూడా ఉపయోగించవచ్చు. ఉదాహరణకు, నేను ఇక్కడ `<br/>` ఉపయోగించాను, తద్వారా లైన్ బ్రేక్ అవుతుంది.

```php
<?php
    echo "ఇది మొదటి వాక్యం<br/>";
    echo "ఇది రెండవ వాక్యం";
    echo "<a href=\"https://example.com\">ఇక్కడ ఒక లింక్ ఉంది</a>";
?>
```

## వేరియబుల్

మనం ఒక వేరియబుల్కు విలువను అసైన్ చేయవచ్చు, ఇది మనకు `PHP` రాయడాన్ని సులభతరం చేస్తుంది, `Javascript` వలె సమానంగా ఉంటుంది. మనం డాలర్ సైన్ `$` ను ఉపయోగిస్తాము.

```php
<?php
    $విలువ = "నేను ఒక విలువ";
?>
```

వేరియబుల్స్ పేరు పెట్టేటప్పుడు మనం అనుసరించాల్సిన కొన్ని నియమాలు ఉన్నాయి.

<టేబుల్
  headers={['ఉదాహరణ', 'వివరణ', 'నియమాలు']}
  rows={[
    ['$విలువ', 'లోవర్ కేస్', '✅'],
    ['$విలువ', 'అప్పర్ కేస్', '✅'],
    ['$_విలువ', 'అండర్ స్కోర్', '✅'],
    ['$1విలువ', 'నంబర్ నుండి ప్రారంభం', '❌']
  ]}
/>

వేరియబుల్ పేర్లలో `అప్పర్ కేస్` మరియు `లోవర్ కేస్` మధ్య తేడా ఉంది. అప్పర్ కేస్ వేరియబుల్స్ లోవర్ కేస్ వేరియబుల్స్కు సమానం కాదు.

## వేరియబుల్ నేమింగ్ టిప్స్

### కేమెల్ కేస్

ఒక వేరియబుల్ పేరు మరియు ఫంక్షన్ పేరు రెండు లేదా అంతకంటే ఎక్కువ పదాలతో ఒక ప్రత్యేకమైన ఐడెంటిఫికేషన్ పదంగా లింక్ చేయబడినప్పుడు, "కేమెల్ కేస్" ఉపయోగం వేరియబుల్స్ మరియు ఫంక్షన్ల రీడబిలిటీని పెంచుతుంది. ఎందుకంటే మనం వేరియబుల్స్లో `స్పేసెస్` ఉంచలేము.

```php
<?php
    $ఒకటిప్లస్ఒకటిజవాబు = 1 + 1;
    echo $ఒకటిప్లస్ఒకటిజవాబు;
?>
```

### స్నేక్ కేస్

స్నేక్ కేస్ (స్నేక్_కేస్ వలె) ఒక రైటింగ్ స్టైల్ను సూచిస్తుంది, దీనిలో ప్రతి స్పేస్ ఒక అండర్ స్కోర్ ( \_ ) క్యారెక్టర్తో భర్తీ చేయబడుతుంది మరియు ప్రతి పదం యొక్క మొదటి అక్షరం లోవర్ కేస్లో వ్రాయబడుతుంది.

```php
<?php
    $ఒకటి_ప్లస్_ఒకటి_జవాబు = 1 + 1;
    echo $ఒకటి_ప్లస్_ఒకటి_జవాబు;
?>
```

## స్ట్రింగ్ ఆపరేటర్

రెండు స్ట్రింగ్ ఆపరేటర్లు ఉన్నాయి. మొదటిది కన్కటినేషన్ ఆపరేటర్ ("."), ఇది దాని ఎడమ మరియు కుడి ఆర్గ్యుమెంట్ల కన్కటినేటెడ్ స్ట్రింగ్ను రిటర్న్ చేస్తుంది. రెండవది కన్కటినేషన్ అసైన్మెంట్ ఆపరేటర్ (".="), ఇది కుడి ఆర్గ్యుమెంట్ను ఎడమ ఆర్గ్యుమెంట్కు అపెండ్ చేస్తుంది.

```php
<?php
    echo "ఇది" . " " . "స్ట్రింగ్" . " " . "ఆపరేటర్స్" . ".";
?>
```

## అరిథ్మెటిక్ ఆపరేటర్

```php
<?php
    $విలువ0 = 1.5 // ఫ్లోట్
    $విలువ1 = 1 + 1 // ప్లస్
    $విలువ2 = 1 - 1 // మైనస్
    $విలువ3 = 1 * 1 // టైమ్స్
    $విలువ4 = 1 / 1 // డివైడ్
    $విలువ5 = 12 / 5 // డివైడ్ | ఇన్కంప్లీట్ డివిజన్ ఫ్లోట్ అవుతుంది, ఇది ఇతర ప్రోగ్రామింగ్ భాషల నుండి భిన్నంగా ఉంటుంది, php రిమైన్డర్ను లెక్కిస్తుంది
    $విలువ6 = 3 % 5 // రిమైన్డర్ తీసుకోండి
?>
```
