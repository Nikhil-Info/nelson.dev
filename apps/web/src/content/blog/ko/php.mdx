## PHP 소개

PHP (전체 이름: PHP: _Hypertext Preprocessor_)는 오픈 소스 범용 스크립팅 언어로, 특히 웹 개발에 적합하며 HTML에 포함될 수 있습니다. PHP 구문은 C, Java, Perl과 같은 인기 있는 프로그래밍 언어의 특성을 차용하며 일반 프로그래머가 쉽게 배울 수 있습니다. PHP의 주요 목표는 웹 개발자가 동적 페이지를 빠르게 작성할 수 있도록 하는 것이지만 PHP는 다른 많은 영역에서도 사용됩니다.

PHP 파일 확장자는 `.php`입니다, 예를 들어 `index.php`. 위에서 언급한 것처럼 `PHP`는 `HTML`을 직접 포함할 수 있습니다.

## 사용법

브라우저가 `<?php`를 위에서 아래로 읽을 때, `PHP`를 실행하고 `?>`로 끝냅니다. 예를 들어:

```php
<!DOCTYPE html>
<html>
    <head>
        <title>PHP</title>
    </head>
    <body>
        <?php
            echo "이 문장입니다."
        ?>
    </body>
</html>
```

## 주의사항

`PHP`에서는 `PHP` 명령어가 끝난 후 다른 `HTML`이 없습니다. `?>`로 끝내지 않는 것이 좋습니다. 닫으면 문제가 발생할 수 있습니다.

```php
<?php
    echo "PHP 종료 후 HTML 없음";
```

## PHP 주석

거의 대부분의 웹 프로그래밍 언어는 주석을 달 수 있으며, `PHP`의 **한 줄** 방식은 `//`이고 **여러 줄** 방식은 `/* 여기에 일부 텍스트 */`입니다.

```php
<?php
    echo = "이것은 한 줄 주석입니다"; // 나는 한 줄 주석입니다
    echo = "이것은 여러 줄 주석입니다";
    /*
        나는 여러 줄 주석입니다
        두 번째 줄
    */
?>
```

## Echo

`PHP`에서 가장 간단한 구문은 `Echo`입니다. `Echo`의 내용은 큰따옴표 `""`나 작은따옴표 `''`로 묶어야 합니다. 각 구문은 세미콜론 `;`으로 끝나야 합니다.

```php
<?php
    echo "이것은 큰따옴표 사용 예시입니다.";
    echo '이것은 작은따옴표 사용 예시입니다.';
?>
```

`Echo`의 내용은 `HTML` 구문도 사용할 수 있습니다. 예를 들어, 여기서는 줄바꿈을 위해 `<br/>`을 사용했습니다.

```php
<?php
    echo "이것은 첫 번째 문장입니다<br/>";
    echo "이것은 두 번째 문장입니다";
    echo "<a href=\"https://example.com\">여기 링크입니다</a>";
?>
```

## 변수

`PHP` 작성을 쉽게 하기 위해 변수에 값을 할당할 수 있으며, `Javascript`와 유사합니다. 달러 기호 `$`를 사용합니다.

```php
<?php
    $value = "나는 값입니다";
?>
```

변수 이름을 지을 때 따라야 할 몇 가지 규칙이 있습니다.

<Table
  headers={['예시', '설명', '규칙']}
  rows={[
    ['$value', '소문자', '✅'],
    ['$Value', '대문자', '✅'],
    ['$_value', '밑줄', '✅'],
    ['$1value', '숫자로 시작', '❌']
  ]}
/>

변수 이름에서 **대문자**와 **소문자**는 차이가 있습니다. 대문자 변수는 소문자 변수와 같지 않습니다.

## 변수 이름 지정 팁

### Camel case

변수 이름과 함수 이름이 두 개 이상의 단어로 연결되어 고유한 식별 단어를 형성할 때, "camel case"를 사용하면 변수와 함수의 가독성을 높일 수 있습니다. 변수에는 `공백`을 가질 수 없기 때문입니다.

```php
<?php
    $onePlusOneAnswer = 1 + 1;
    echo $onePlusOneAnswer;
?>
```

### Snake case

Snake case (snake_case처럼)는 각 공백을 밑줄( _ )로 대체하고 각 단어의 첫 글자를 소문자로 쓰는 작성 스타일을 말합니다.

```php
<?php
    $one_plus_one_answer = 1 + 1;
    echo $one_plus_one_answer;
?>
```

## 문자열 연산자

두 개의 문자열 연산자가 있습니다. 첫 번째는 연결 연산자(".")로, 왼쪽과 오른쪽 인수의 연결된 문자열을 반환합니다. 두 번째는 연결 할당 연산자(".=")로, 오른쪽 인수를 왼쪽 인수에 추가합니다.

```php
<?php
    echo "이것은" . " " . "문자열" . " " . "연산자" . "입니다.";
?>
```

## 산술 연산자

```php
<?php
    $value0 = 1.5 // Float
    $value1 = 1 + 1 // 더하기
    $value2 = 1 - 1 // 빼기
    $value3 = 1 * 1 // 곱하기
    $value4 = 1 / 1 // 나누기
    $value5 = 12 / 5 // 나누기 | 불완전한 나눗셈은 float가 되며, 다른 프로그래밍 언어와 다르게 php는 나머지를 계산함
    $value6 = 3 % 5 // 나머지 구하기
?>
```
