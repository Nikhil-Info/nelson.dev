## Introducción de PHP

PHP (nombre completo: PHP: _Hypertext Preprocessor_) es un lenguaje de scripting de computadora de código abierto de propósito general, especialmente adecuado para desarrollo web e incrustado en HTML. La sintaxis de PHP toma prestadas características de lenguajes de computadora populares como C, Java y Perl, y es fácil de aprender para programadores generales. El objetivo principal de PHP es permitir a los desarrolladores web escribir rápidamente páginas dinámicas, pero PHP también se usa en muchas otras áreas.

La extensión de archivo PHP es `.php`, por ejemplo `index.php`. Como se mencionó anteriormente, `PHP` puede incrustar directamente `HTML`.

## Uso

Cuando el navegador lee `<?php` de arriba hacia abajo, ejecuta `PHP` y termina con `?>`. Por ejemplo:

```php
<!DOCTYPE html>
<html>
    <head>
        <title>PHP</title>
    </head>
    <body>
        <?php
            echo "Esta es una oración."
        ?>
    </body>
</html>
```

## Precauciones

En `PHP`, no hay otro `HTML` después de que se complete la instrucción `PHP`. Se recomienda no terminar con `?>`. Causará problemas si se cierra.

```php
<?php
    echo "No hay HTML después de que PHP termina";
```

## Comentarios de PHP

Casi la mayoría de los lenguajes de programación web pueden comentar, y la forma **de una línea** de `PHP` es `//` y la forma **multilínea** es `/* Algún texto aquí */`.

```php
<?php
    echo = "Este es un comentario de una línea"; // Soy un comentario de una línea
    echo = "Este es un comentario multilínea";
    /*
        Soy un comentario multilínea
        Segunda línea
    */
?>
```

## Echo

La sintaxis más simple en `PHP` es `Echo`. El contenido de `Echo` debe estar entre comillas dobles `""` o comillas simples `''`. Cada sintaxis debe terminar con un punto y coma `;`.

```php
<?php
    echo "Este es el ejemplo usando comillas dobles.";
    echo 'Este es el ejemplo usando comillas simples.';
?>
```

El contenido en `Echo` también puede usar sintaxis `HTML`. Por ejemplo, usé `<br/>` aquí, para que la línea pueda romperse.

```php
<?php
    echo "Esta es la primera oración<br/>";
    echo "Esta es la segunda oración";
    echo "<a href=\"https://example.com\">aquí hay un enlace</a>";
?>
```

## Variable

Podemos asignar un valor a una variable para facilitarnos la escritura de `PHP`, similar a `Javascript`. Usaremos el signo de dólar `$`.

```php
<?php
    $valor = "Soy un valor";
?>
```

Hay algunas reglas a seguir cuando nombramos variables.

Hay una diferencia entre `mayúsculas` y `minúsculas` en los nombres de variables. Las variables en mayúsculas no son iguales a las variables en minúsculas.

## Consejos para nombrar variables

### Camel case

Cuando un nombre de variable y un nombre de función están vinculados por dos o más palabras para formar una palabra de identificación única, el uso de "camel case" puede aumentar la legibilidad de variables y funciones. Porque no podemos tener `espacios` en las variables.

```php
<?php
    $unoMasUnoRespuesta = 1 + 1;
    echo $unoMasUnoRespuesta;
?>
```

### Snake case

Snake case (como snake_case) se refiere a un estilo de escritura en el que cada espacio se reemplaza con un carácter de guión bajo ( \_ ) y la primera letra de cada palabra se escribe en minúsculas.

```php
<?php
    $uno_mas_uno_respuesta = 1 + 1;
    echo $uno_mas_uno_respuesta;
?>
```

## Operador de cadena

Hay dos operadores de cadena. El primero es el operador de concatenación ("."), que devuelve la cadena concatenada de sus argumentos izquierdo y derecho. El segundo es el operador de asignación de concatenación (".="), que agrega el argumento derecho al argumento izquierdo.

```php
<?php
    echo "Esto" . " " . "es" . " " . "operadores" . " " . "de" . " " . "cadena" . ".";
?>
```

## Operador aritmético

```php
<?php
    $valor0 = 1.5 // Flotante
    $valor1 = 1 + 1 // Más
    $valor2 = 1 - 1 // Menos
    $valor3 = 1 * 1 // Por
    $valor4 = 1 / 1 // Dividir
    $valor5 = 12 / 5 // Dividir | La división incompleta se convertirá en flotante, lo cual es diferente de otros lenguajes de programación, php calculará el resto
    $valor6 = 3 % 5 // Tomar el resto
?>
```
