## PHPの紹介

PHP（正式名称：PHP: _Hypertext Preprocessor_）は、オープンソースの汎用コンピュータスクリプト言語で、特にWeb開発に適しており、HTMLに埋め込むことができます。PHPの構文はC言語、Java、Perlなどの人気のあるコンピュータ言語の特徴を取り入れており、一般的なプログラマーが学びやすいです。PHPの主な目標は、Web開発者が迅速に動的ページを書くことを可能にすることですが、PHPは他の多くの分野でも使用されています。

PHPファイルの拡張子は`.php`で、例`index.php`。上記のように、`PHP`は直接`HTML`を埋め込むことができます。

## 使用方法

ブラウザが上から下に`<?php`を読み取ると、`PHP`を実行し、`?>`で終了します。例：

```php
<!DOCTYPE html>
<html>
    <head>
        <title>PHP</title>
    </head>
    <body>
        <?php
            echo "これは文です。"
        ?>
    </body>
</html>
```

## 注意事項

`PHP`では、`PHP`命令が完了した後に他の`HTML`はありません。`?>`で終了しないことをお勧めします。閉じると問題が発生します。

```php
<?php
    echo "PHP終了後にHTMLはありません";
```

## PHPコメント

ほとんどのWebプログラミング言語はコメントできますが、`PHP`の**単一行**の方法は`//`で、**複数行**の方法は`/* ここにテキスト */`です。

```php
<?php
    echo = "これは単一行コメントです"; // 私は単一行コメントです
    echo = "これは複数行コメントです";
    /*
        私は複数行コメントです
        2行目
    */
?>
```

## エコー

`PHP`で最も簡単な構文は`Echo`です。`Echo`の内容は二重引用符`""`または単一引用符`''`で囲む必要があります。各構文はセミコロン`;`で終了する必要があります。

```php
<?php
    echo "これは二重引用符を使用する例です。";
    echo 'これは単一引用符を使用する例です。';
?>
```

`Echo`内のコンテンツは`HTML`構文も使用できます。たとえば、ここで`<br/>`を使用したので、行を改行できます。

```php
<?php
    echo "これは最初の文です<br/>";
    echo "これは2番目の文です";
    echo "<a href=\"https://example.com\">こちらがリンクです</a>";
?>
```

## 変数

`Javascript`と同様に、変数に値を割り当てて`PHP`を簡単に書くことができます。ドル記号`$`を使用します。

```php
<?php
    $値 = "私は値です";
?>
```

変数に名前を付ける際に従うべきいくつかのルールがあります。

<表
  headers={['例', '説明', 'ルール']}
  rows={[
    ['$値', '小文字', '✅'],
    ['$値', '大文字', '✅'],
    ['$_値', 'アンダースコア', '✅'],
    ['$1値', '数字から開始', '❌']
  ]}
/>

変数名の`大文字`と`小文字`には違いがあります。大文字の変数は小文字の変数と等しくありません。

## 変数命名のヒント

### キャメルケース

変数名と関数名が2つ以上の単語で結びついて一意の識別語を形成する場合、「キャメルケース」を使用すると、変数と関数の読みやすさが向上します。変数に`スペース`を含めることができないためです。

```php
<?php
    $一足す一答え = 1 + 1;
    echo $一足す一答え;
?>
```

### スネークケース

スネークケース（snake_caseのように）は、すべてのスペースがアンダースコア（ \_ ）文字に置き換えられ、すべての単語の最初の文字が小文字で書かれる書き方を指します。

```php
<?php
    $一_足す_一_答え = 1 + 1;
    echo $一_足す_一_答え;
?>
```

## 文字列演算子

2つの文字列演算子があります。1つ目は連結演算子（"."）で、左辺と右辺の引数を連結した文字列を返します。2つ目は連結代入演算子（".="）で、右辺の引数を左辺の引数に追加します。

```php
<?php
    echo "これ" . " " . "は" . " " . "文字列" . " " . "演算子" . " " . "です" . "。";
?>
```

## 算術演算子

```php
<?php
    $値0 = 1.5 // 浮動小数点数
    $値1 = 1 + 1 // 加算
    $値2 = 1 - 1 // 減算
    $値3 = 1 * 1 // 乗算
    $値4 = 1 / 1 // 除算
    $値5 = 12 / 5 // 除算 | 不完全な除算は浮動小数点数になります。これは他のプログラミング言語とは異なり、phpは余りを計算します
    $値6 = 3 % 5 // 余りを取る
?>
```
